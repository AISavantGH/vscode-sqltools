> SQLTools driver generator

## Installation

First, install [Yeoman](http://yeoman.io) and `generator-sqltools-driver` using `npm` or `yarn`.

```bash
npm install -g yo
npm install -g generator-sqltools-driver
```

Then generate your new project:

```bash
yo sqltools-driver
```
